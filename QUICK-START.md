# ⚡ Quick Start - AI Code Generation in 5 Minutes

## 🚀 Super Quick Setup

### **Step 1: Run Installation Script** (2 minutes)

**Mac/Linux:**
```bash
chmod +x install-ai-tools.sh
./install-ai-tools.sh
```

**Windows:**
```cmd
install-ai-tools.bat
```

### **Step 2: Restart VS Code** (1 minute)
- Close VS Code completely
- Reopen VS Code

### **Step 3: Sign Up (2 minutes)**

**Codeium** (Recommended):
1. Click Codeium icon in VS Code sidebar
2. Click "Sign Up"
3. Sign in with GitHub
4. Verify email
5. Done!

**That's it! AI is now working! ✅**

## 🆘 Troubleshooting

### AI not showing suggestions?
```bash
# In VS Code terminal:
code --list-extensions | grep -i codeium
```

If no output, reinstall:
```bash
code --install-extension Codeium.codeium
```

### Still not working?
1. Restart VS Code
2. Check your internet connection
3. Sign in to Codeium again
4. Look at: View → Output → Codeium
